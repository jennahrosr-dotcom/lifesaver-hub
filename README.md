# LifeSaver Hub 🩸
### UiTM Jasin Blood Donation Management System

A web-based blood donation management system developed to streamline and digitise the blood donation process at UiTM Jasin. This project was published at IEEE SCOReD 2025.

---

## Features

- **Account Management** — Registration and login for donors and administrators
- **Event Management** — Admins can create, update, and manage donation events
- **Donation Application** — Donors can apply to participate in upcoming events
- **Real-time Health Eligibility Questionnaire** — Instant feedback on donor eligibility based on health criteria
- **Attendance Confirmation** — Track and confirm donor attendance at events
- **Gamified Reward Tracker** — Achievement badges and points to encourage repeat donations
- **Automated Email Notifications** — Multi-trigger email system for new events, event changes, registration confirmations, and day-of reminders
- **Report Generation** — Admins can generate donation reports
